# PreciousMetalInvestment

这是一个用于贵金属投资研究与风险监测的个人项目，核心目标不是预测每日涨跌，而是持续回答两个问题：

1. 黄金/贵金属长期牛市判断是否仍然成立。
2. 当前回调是牛市中继、右侧转折前震荡，还是牛市 thesis 已经需要重审。

项目分为两条线：一条维护中长期理论框架，另一条把框架转化为定期更新的市场信号仪表盘。

## 项目内容

### 研究框架

- `research/黄金中长期趋势研究框架（持续更新版）.md`
  - 当前持续更新的主研究文件。
  - 用于沉淀定价逻辑、宏观情景、可证伪条件和方法论纪律。

- `research/黄金中长期趋势研究框架 v2.md`
  - 原始冻结归档版。
  - 不随讨论更新，用作未来回溯和约束基线。

- `research/patches/`
  - 保存对研究框架的结构性补丁来源。

### 市场监测

- `dashboard.md`
  - 最新一期贵金属牛市仪表盘。
  - 用红/黄/蓝/绿等状态判断牛市完整性、回调阶段和仓位风险。

- `dashboard_plan.md`
  - 仪表盘规则口径。
  - 用来避免每次根据行情临场改叙事。

- `data/indicators.csv`
  - 指标历史记录。

- `data/event_log.csv`
  - 关键事件后的金价反应日志。

- `reports/`
  - 阶段性市场分析报告。

## 当前框架重点

研究框架把黄金行情拆成三个时钟：

- 长期论题：财政主导、金融抑制、去美元化、央行和主权配置。
- 行情时钟：实际利率、ETF资金流、COT仓位、就业裂痕、油价冲击。
- 行动纪律：何时保持观察、何时允许增量资金、何时必须重审牛市 thesis。

持续更新版已经加入“底仓分量与硬核层”补丁：结构性买盘不再被默认视为已经验证的地板，而是需要在深度回调中通过需求响应事后检验。也就是说，`$3,800` 不是天然支撑，而是一个待检验区域。

## 使用方式

阅读顺序建议：

1. 先读 `research/黄金中长期趋势研究框架（持续更新版）.md` 理解框架。
2. 再读 `dashboard_plan.md` 理解仪表盘规则。
3. 查看 `dashboard.md` 获取最新市场状态。
4. 使用 `data/indicators.csv` 和 `data/event_log.csv` 复盘信号变化。
